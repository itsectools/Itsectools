# Network Pulse — Complete Guide

> **URL:** [itsectools.com/network-pulse](https://itsectools.com/network-pulse)

## Overview

Real-time network telemetry, latency analysis, and connection health monitoring with AI-driven security insights.

---

## Metrics

| Metric | What It Measures |
|---|---|
| **Public IP** | Your external IP address as seen by the internet |
| **Nearest Edge Server** | Your closest CDN/edge node and geographic location |
| **Latency (RTT)** | Round-trip time to the server in milliseconds |
| **Jitter** | Variance in latency — indicates connection stability |
| **Packet Loss** | Percentage of packets lost during transmission |
| **PMTU Discovery** | Path Maximum Transmission Unit via binary search algorithm |
| **Connection Quality** | Overall score based on all metrics combined |

---

## AI Security Insights

After collecting telemetry, Network Pulse generates automated security recommendations based on your network performance. These insights highlight potential security implications of your network characteristics — for example:

- High latency suggesting traffic inspection overhead
- Jitter patterns indicating network congestion or DDoS
- Edge server location and routing path analysis
- MTU issues that may affect VPN or tunnel performance
